Pasos para crear los servidores:

Las diferentes configuraciones las hemos realizado en una máquina virtual con Debian 10, para configurar todo el entorno de trabajo para que funcione correctamente el Caso de Estudio de Europa.

Para cada una de estas secciones es necesario realizar antes una actualizacion de paquetes.

sudo apt-get update

Instale el paquete de código abierto de NGINX.

sudo apt-get install nginx openssl

Verficacion del servicio habilitando NGNIX.

sudo systemctl enable nginx

Habilitar UFW lo cual el siguiente comando el cual carga el firewall y le permite iniciarse al arrancar el sistema.

sudo ufw enable

Habilitar otras conexiones, para abrir los puertos 80 y 433.

sudo ufw allow 'Nginx Full'

Revisamos el estado de NGINX y del Firewall.

systemctl status nginx sudo ufw status

Copia el directorio

sudo cp /etc/nginx/sites-available/default /etc/nginx/sites-available/europa.conf

Borramos el directorio.

sudo rm -f /etc/nginx/sites-enabled/default

Enlace el siguiente directorio.

sudo ln -s /etc/nginx/sites-available/europa.conf /etc/nginx/sites-enabled/

Creamos el certificado

mkdir /etc/nginx/certificate cd /etc/nginx/certificate openssl req -new -newkey rsa:4096 -x509 -sha256 -days 365 -nodes -out nginx-certificate.crt -keyout nginx.key

Editamos el archivo europa.conf

sudo nano /etc/nginx/sites-available/europa.conf

Guardamosel archivo ctrl + X -> S ->Enter y Reiniciamos el servicio

sudo systemctl restart nginx

Zona Inversa

Validamos la sintaxis del archi de configuración BIND.

Activar Zona Directa

sudo named-checkzone europa.net /etc/bind/forward.europa.net.db

Activar Zona Inversa

sudo named-checkzone 1.168.192.in-addr.arpa /etc/bind/reverse.europa.net.db

Reiniciamos el servicio de bind

sudo systemctl restart bind9

Verificamos el estado del servicio

sudo systemctl status bind9

Maquina Cliente

Vaya a cualquier máquina cliente y agregue nuestra nueva dirección IP del servidor DNS en el archivo /etc/resolv.conf.

Utilice el comando dig para verificar la búsqueda directa

dig www.europa.net

Confirme la búsqueda inversa con el comando dig.

dig -x 192.168.0.3
